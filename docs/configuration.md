# Настройки

Все настройки доступны в боковой панели и сохраняются в `localStorage` между сессиями.

## Платформы

| Платформа | Лимит символов | Особенности |
|-----------|----------------|-------------|
| Twitter/X | 280 | Без длинных абзацев, максимум концентрация |
| Instagram | 2 200 | Визуальный storytelling, эмодзи |
| LinkedIn | 3 000 | Профессиональный тон, без жаргона |
| Facebook | Без лимита | Дружелюбный тон, широкая аудитория |
| TikTok | 2 200 | Молодёжная аудитория, вирусный контент |
| Telegram | Без лимита | Информативный стиль, Markdown-дружелюбен |

Счётчик символов в результате показывает предупреждение при превышении лимита платформы.

## Тон

| Тон | Описание |
|-----|----------|
| Профессиональный | Деловой, авторитетный, без жаргона |
| Casual | Разговорный, тёплый, близкий к аудитории |
| Юмористический | Смешной, игривый, с каламбурами |
| Вдохновляющий | Мотивирующий, поднимает боевой дух |
| Смелый | Дерзкий, провокационный, запоминающийся |

## Длина

| Длина | Описание |
|-------|----------|
| Короткий | 1–3 предложения — идеально для Twitter/X |
| Средний | 3–7 предложений — универсальный вариант |
| Длинный | 7+ предложений — развёрнутый пост для LinkedIn |

## Хештеги и эмодзи

- **Хештеги** — при включении Claude добавляет релевантные хештеги в конец поста
- **Эмодзи** — при включении эмодзи используются уместно для усиления текста

## Язык ответа

Поле **Язык** в боковой панели задаёт язык генерируемого поста.  
Примеры: `Русский`, `English`, `Deutsch`, `Español`.

Язык интерфейса PostCraft фиксирован (русский) и не зависит от этой настройки.

## Где хранятся настройки

Настройки (`platform`, `tone`, `length`, `includeHashtags`, `includeEmojis`, `language`) сохраняются в Zustand store с persist middleware:

```
localStorage → ключ "postcraft-store"
               поля: apiKey, settings
               (история сообщений не сохраняется)
```

Для сброса: `DevTools → Application → Local Storage → postcraft-store → Delete`.

## Системный промпт

При каждом запросе настройки компонуются в системный промпт для Claude:

```
Ты — профессиональный SMM-копирайтер мирового уровня.

Платформа: Instagram (до 2200 символов, визуальный storytelling)
Тон: непринуждённый и дружелюбный
Длина: средний (3-7 предложений)
Хештеги: добавь релевантные хештеги
Эмодзи: используй уместно
Язык ответа: Русский

Правила: писать ТОЛЬКО готовый пост, без вступлений.
```
